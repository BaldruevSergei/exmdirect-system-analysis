EXMDirect — System Analysis Case
Обзор
EXMDirect — аналитический кейс проектирования workflow-системы с управлением жизненным циклом операций, формализованной валидацией данных и интеграцией через REST API.
Предметная область системы — образовательное тестирование.
Однако архитектурно проект моделирует структурированный процесс обработки сценариев с управлением статусами, транзакционными сущностями и контролем целостности данных.
Система включает:
формализованный формат входных данных (DSL-подобная структура тестов),
автоматическую валидацию и расчёт результатов,
управление жизненным циклом попыток (Attempt lifecycle),
идентификацию пользователей через QR,
интеграцию с Telegram Bot API,
REST API контракт (OpenAPI).

Архитектурный фокус
В рамках проекта продемонстрированы:
моделирование бизнес-процессов (BPMN),
управление статусами операций,
проектирование транзакционных сущностей (Attempt / Result),
структурированная валидация входных данных,
контрактное проектирование API (OpenAPI),
интеграция с внешним каналом уведомлений,
ролевая модель доступа.
Данные принципы применимы в системах:
обработки заявок,
онбординга клиентов,
rule-based системах (скоринг, проверки),
системах с управлением жизненным циклом операций.
